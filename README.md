# FacePrep SRMIST Login

A FacePrep-inspired login page for SRMIST students.

## Features
- Email validation for @srmist.edu.in
- Modern glassmorphism UI with FacePrep colors
- Admin panel for viewing login attempts
- Redirects to faceprep.in after login

## Local Development
```bash
python server.py
```

## Deployment
Deployed on Vercel - credentials are logged to Vercel logs.
